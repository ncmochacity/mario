What is keydown() function? What does it do?

$('DOM selector').keydown()binds an event handler passed in a function keydown.(function(event){event.which==37}) determining which key was pressed. event.which property can get the keycode corresponding to keys pressed on the keyboard like "37, 38, 39,40, 13...etc".
